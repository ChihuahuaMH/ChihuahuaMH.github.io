Introduction to Kotlin

Kotlin is the recommended programming language for developing Android apps.

Program as series of instructions for the system to perform some action. For example, write an instruction to print Hello, World! :

fun main() {
    println("Hello, world!")
}

Using keyword val to define read-only variables.

val score = 90
println("You got ${score}!")

var keyword can reassign variables.

var num = 5
num += 10

📣 Note

  1. All Kotlin program need to have a main() function : fun main() {}.

  2. println() print the text with adding line break at the end of string, while print() not.

User Interface (UI)

  • View

    Each of elements is called View.

  • ViewGroup

    It is a container that View objects can sit in,and is responsible for arranging the Views inside it.

  • TextView

    View that is for displaying text.

  • ImageView

    Displaying Image in app. ImageViews should have a content description to help app more accessible.

    Use setImageResource() to change image.

Hardcoded String

  • It is written directly in the code of app.
  • Difficult to translate app into other languages.
  • Hard to reuse a string.

Data types

  • Int

  • String

  • Boolean

  • IntRange

    1..6 is represent the range of int from 1 to 6. Call random() on an IntRange to generate a number.

when statement

Kotlin has when statement to make else if easier to write and read.

// if-else
if (condition-is-true) {
    execute-this-code
} else if (condition-is-true) {
    execute-this-code
} else {
    execute-this-code
}

// when
when (variable) {
    matches-value -> execute-this-code
    matches-value -> execute-this-code
    ...
}

Activity provides the window that app draws its UI. Every app has one or more activities.

Testing is a structure method of checking software to make sure it works correctly. For example:

class ExampleUnitTest {
   @Test
   fun addition_isCorrect() {
       assertEquals(4, 2 + 3)
   }
}
