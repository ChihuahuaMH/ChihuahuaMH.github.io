[GitHub] 解決 remote: Invalid username or password 問題

August 10, 2022 · 1 min · Min-Hsien
Table of Contents
  • 問題
  • 解決方法
  • 參考資料

問題#

前陣子在更新部落格後，使用 deploy.sh 會出現以下問題：

remote: Invalid username or password.
fatal: Authentication failed for 'https://github.com/[user]/[repo].git/'

查了資料後發現和 Personal Access Token 有關，立刻跑去看～竟然過期了！！！

解決方法#

Token 過期的話，產生新的 Personal Access Token 就好了！

基本上和之前 [Hugo] 自動部署網站到 GitHub 與 GitHub Pages 生成 Token 的方式一樣，要記得先將原本過期的 Token 刪除～

接著加入 Actions secrets 的部分，不需要刪除，按 Update 把剛剛拿到的值貼進去就好了！
